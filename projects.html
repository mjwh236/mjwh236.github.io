Maze Generation with Solver

This project was one of the more difficult asignments from from Computer Graphics Class. This project was created in java. When generating the maze a randomized depth-first search is preformed from each square of the grid to create the walls of the maze. The dimensions of the maze can be set indepenantly and by the user. The maze generation can both be stopped and resumed via buttons. The speed of solving the maze can be adjusted using the slider.
